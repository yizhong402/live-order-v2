# 直播订单管理系统 V2

精简版直播订单管理工具，纯手动触发同步，零守护进程。

## 技术栈

- 原生 HTML + CSS + JS，零框架依赖
- BaaS 后端 (baas.kuafuai.net)
- OMS OAuth 2.0 三步授权
- GitHub Pages 部署

## 模块

| 模块 | 说明 |
|------|------|
| 🏠 首页 | 库存概览、活跃场次、低库存预警、热卖预览、最近订单 |
| 📝 订单录入 | SKU扫码、竞拍价、轮次管理、超卖检测、实时订单列表 |
| 📦 商品管理 | 商品列表、搜索筛选、手动同步OMS、库存编辑 |
| 🔗 组合SKU | 创建/编辑组合、子SKU管理、扫码自动展开 |
| 🔥 热卖排行 | 全量订单聚合Top 100、搜索导出 |
| 🆕 新上架 | 全量上架列表、日期筛选、售罄补货标记 |
| 📜 历史记录 | 场次日期筛选、订单详情展开、CSV导出 |
| 💰 毛利计算 | 收入/成本/毛利/毛利率、SKU成本明细 |
| ⚙️ 系统设置 | OMS OAuth配置、手动同步、低库存阈值、数据管理 |

## 部署

推送至 `main` 分支，GitHub Pages 自动部署。

## 开发

```bash
cd live-order-v2
python3 -m http.server 8086
# 访问 http://localhost:8086
```
